choose/find 3 less common (weird?) creative technology tools

Blynk
https://blynk.io/
Blynk is a hardware-agnostic IoT platform with white-label mobile apps, private clouds, device management, data analytics, and machine learning.

vvvv
https://vvvv.org/
vvvv is a general purpose toolkit with a special focus on real-time video synthesis and programming large media environments with physical interfaces, real-time motion graphics, audio and video. vvvv uses a dataflow approach and a visual programming interface for rapid prototyping and developing.

Pure Data
https://puredata.info/
Pure Data (Pd) is a visual programming language developed by Miller Puckette in the 1990s for creating interactive computer music and multimedia works. While Puckette is the main author of the program, Pd is an open-source project with a large developer base working on new extensions. It is released under a license similar to the BSD license. It runs on GNU/Linux, Mac OS X, iOS, Android and Windows. Ports exist for FreeBSD and IRIX.

THE OBJECT MARRIAGE
(https://www.youtube.com/watch?v=tWwPd_wtg68&t=6s)
Celine Park
https://www.celineparkgallery.com/blank-page-1
The project is a 20-minute performance directed and designed by Céline Park. It is a collaborative project with various artists and a scientist. The project speculates a fictional scenario of ‘What if … marriage to objects conformed more strictly to the societal rules?’ In this project, Céline has imagined a dystopian society where objectophiles can only marry the objects they love once they are qualified.

ETERNAL RECURRENCE
(https://www.youtube.com/watch?time_continue=7&v=SZTTPS0pgQI&feature=emb_logo)
SOGANG MEDIA LAB(Professor Moon-Ryul Jung)
Clouds become droplets, droplets fall on the ground and flow, the flowing water evaporates, and the vapor forms clouds to start yet another cycle. This ever-repeating cycle of water can be expressed with the phrase “Eternal Recurrence.” This piece borrows the eternal recurrence of water drops to express the cycle of human life in which one is born to the world, advances through life, seeks true freedom and happiness that transcend the confines of reality, and tries to leave beautiful memories of that life. =======
